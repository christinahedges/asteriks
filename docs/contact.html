Contact us


If you have any problems using out data or getting asteriks to work, you can contact us in a few different ways. .



Open an issue on github

If you are unable to run asteriks or believe there is a problem with one of the asteriks data products, try opening an issue on the asteriks repository. You can do this by clicking the issues button on the asteriks repository and creating a new issue. (You will need a github account). This is the fastest way to get in touch, and has the added benefit of being completely open!

Email directly

Alternately you can email us directly at the Kepler/K2 Guest Observer office at keplergo@mail.arc.nasa.gov. You can find more contact details at the Kepler/K2 Guest Observer website. Questions about K2 Moving Bodies should be directed to Christina Hedges.
