K2 Moving Bodies

Light curves of moving objects from the Kepler/K2 Mission.



The K2 Moving Bodies Project

K2 Moving Bodies is a repository of light curves for asteroids, comets, moons and planets in the solar system observed by Kepler/K2. These light curves have been generated using a procedure described in this paper. You can use the search feature above to find light curves and movies of the moving objects in the Campaigns 1-13. There is also a FAQ for common questions when using K2 Moving Bodies data products. If you have any questions then please contact us.
