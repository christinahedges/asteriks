999


999 (a.k.a. Pluto, 999) is a moving object from K2 campaign 7. You can read more information about this object at the JPL Small-Body Database Browser here. Data was taken from 04 October 2015 to 26 December 2015.

999 was proposed for by Benecchi in GO7012. If you use this data, please cite their proposal. You can find the bibtex citation by clicking the button below.

If only want the light curve of the object with the optimal aperture, download this product. This will give you one .fits file with several extensions.
