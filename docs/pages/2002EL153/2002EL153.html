2002 EL153


2002 EL153 (a.k.a. 57920) is a moving object from K2 campaign 111, 112. You can read more information about this object at the JPL Small-Body Database Browser here. Data was taken from 24 September 2016 to 05 December 2016.

2002 EL153 was proposed for by Pal, Parker, Ryan in GO11114, GO11100, GO11051. If you use this data, please cite their proposal. You can find the bibtex citation by clicking the button below.
