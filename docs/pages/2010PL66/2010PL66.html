2010 PL66


2010 PL66 (a.k.a. 499522) is a moving object from K2 campaign 12. You can read more information about this object at the JPL Small-Body Database Browser here. Data was taken from 15 December 2016 to 04 March 2017.

2010 PL66 was proposed for by Pal in GO12114. If you use this data, please cite their proposal. You can find the bibtex citation by clicking the button below.
